Absolut! Hier ist eine Vorschau des grundlegenden Codes für eine einfache Webseite. Ich habe ihn in die drei Hauptkomponenten aufgeteilt: HTML für die Struktur, CSS für das Design und JavaScript für die Interaktivität. 1. HTML (HyperText Markup Language) - Das Skelett der Seite Dieser Code definiert die grundlegende Struktur und den Inhalt deiner Webseite. <!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meine Webseite</title> <link rel="stylesheet" href="style.css"> </head> <body> <header> <h1>Willkommen auf meiner Webseite</h1> </header> <nav> <a href="#home">Home</a> <a href="#about">Über mich</a> <a href="#contact">Kontakt</a> </nav> <main> <section id="home"> <h2>Startseite</h2> <p>Das ist der Hauptinhalt meiner fantastischen neuen Webseite.</p> </section> <section id="about"> <h2>Über mich</h2> <p>Hier könnten Informationen über dich stehen.</p> </section> </main> <footer> <p>&copy; 2025 Dein Name</p> </footer> <script src="script.js"></script> </body> </html> <!DOCTYPE html> : Definiert den Dokumententyp. <head> : Enthält Metadaten wie den Titel ( <title> ) und die Verknüpfung zur CSS-Datei ( <link> ). <body> : Beinhaltet den gesamten sichtbaren Inhalt der Seite. <header> , <nav> , <main> , <footer> : Strukturieren den Inhalt semantisch in Kopfbereich, Navigation, Hauptinhalt und Fußzeile. 2. CSS (Cascading Style Sheets) - Das Aussehen Dieser Code (in einer Datei namens style.css ) gestaltet die HTML-Elemente. Er sorgt für Farben, Schriftarten und das Layout. /* Allgemeine Stile für den Body */ body { font-family: sans-serif; line-height: 1.6; margin: 0; padding: 0; background-color: #f4f4f4; color: #333; } /* Stil für den Header */ header { background-color: #333; color: #fff; padding: 1rem 0; text-align: center; } /* Stil für die Navigation */ nav { text-align: center; padding: 1rem; background: #444; } nav a { color: #fff; text-decoration: none; padding: 0 15px; } nav a:hover { color: #f4a261; /* Farbe ändert sich beim Darüberfahren */ } /* Hauptinhaltsbereich */ main { padding: 20px; } section { margin-bottom: 20px; } /* Stil für den Footer */ footer { text-align: center; padding: 1rem; background: #333; color: #fff; position: fixed; /* Positioniert den Footer am unteren Rand */ bottom: 0; width: 100%; } body : Legt die grundlegende Schriftart und Farben für die gesamte Seite fest. header , nav , footer : Weisen den Strukturelementen Hintergrundfarben und Textausrichtungen zu. nav a:hover : Ein Beispiel für eine Pseudo-Klasse , die den Stil eines Links ändert, wenn der Mauszeiger darüber schwebt. 3. JavaScript - Die Interaktivität Dieser Code (in einer Datei namens script.js ) kann verwendet werden, um die Seite dynamisch zu machen, z. B. um auf Klicks zu reagieren oder Inhalte zu ändern, ohne die Seite neu zu laden. Hier ist ein einfaches Beispiel: // Wir warten, bis das gesamte HTML-Dokument geladen ist document.addEventListener('DOMContentLoaded', (event) => { // Eine einfache Begrüßungsnachricht in der Konsole des Browsers console.log('Webseite erfolgreich geladen!'); // Beispiel: Eine Nachricht anzeigen, wenn ein Navigationslink geklickt wird const navLinks = document.querySelectorAll('nav a'); navLinks.forEach(link => { link.addEventListener('click', () => { alert('Du hast auf einen Link geklickt!'); }); }); }); document.addEventListener('DOMContentLoaded', ...) : Stellt sicher, dass der Code erst ausgeführt wird, wenn die Seite vollständig geladen ist. document.querySelectorAll('nav a') : Wählt alle Link-Elemente ( <a> ) innerhalb der Navigation ( <nav> ) aus. addEventListener('click', ...) : Fügt jedem dieser Links eine Funktion hinzu, die bei einem Klick eine alert -Nachricht anzeigt. Diese drei Dateien ( index.html , style.css , script.js ) bilden zusammen das Fundament für eine moderne Webseite. Von hier aus kann man dann spezifischere Funktionen und Inhalte hinzufügen.
