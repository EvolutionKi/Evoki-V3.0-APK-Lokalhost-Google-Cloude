Die gängigsten sind: PowerShell (Standard unter Windows) Command Prompt (CMD, ebenfalls Windows) Git Bash (wenn Git installiert ist) WSL (Windows Subsystem for Linux) (wenn du Linux-Distributionen unter Windows nutzt) Welches Terminal solltest du verwenden? Für unsere Zwecke, um Python-Skripte auszuführen und Umgebungsvariablen zu setzen, empfehle ich dir PowerShell . Warum PowerShell? Es ist das Standard-Terminal unter Windows und bietet eine moderne Befehlszeilenumgebung. Der Befehl zum Setzen der Umgebungsvariable ( $env:GEMINI_API_KEY="DEIN_API_KEY_HIER" ) funktioniert direkt in PowerShell. Python-Befehle wie python gemini.py funktionieren dort ebenfalls einwandfrei. So wählst du PowerShell in VS Code aus: Öffne VS Code. Gehe oben im Menü auf Terminal -> Neues Terminal . Unten im Terminal-Fenster siehst du oft einen Dropdown-Pfeil (manchmal auch ein + -Symbol oder den Namen des aktuell geöffneten Terminals). Klicke darauf. Es sollte sich eine Liste mit den verfügbaren Shells öffnen. Wähle dort PowerShell aus.
